# Hermes Agent Skills

凭安智能体技能仓库。通过 GitHub Tap 即可安装使用。

## 安装

```bash
hermes skills tap add github.com/pingan-ai/hermes-skills
hermes skills install enterprise-deep-dive-report
```

## 技能列表

### enterprise-deep-dive-report (v4.0.0)
投行级企业深度研究报告生成器。基于 Doris DAAS（4.08亿企业）+ 32类MCP API + 上市公司对标数据，自动生成八部结构专业投资研究报告。

**触发词：** 企业分析报告、投行报告、企业尽调、深度研究报告、投资研究报告

**依赖：** doris-daas-query
